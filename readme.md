# A simple voxel engine

### Features

- [x] 4x4 Atlas with 4 block textures currently  

- [x] seperate texture for each face  

- [x] mesh generation and face culling for chunk

- [x] Fly fps camera

- [x] Dynamically generate chunks around camera position
